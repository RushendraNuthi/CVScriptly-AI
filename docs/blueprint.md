# **App Name**: CVScriptly AI

## Core Features:

- Multi-Stage Resume Input: Guided form with multiple sections: Personal Info, Education, Experience, etc.
- ATS-Friendly Resume Templates: Selection of resume templates optimized for Applicant Tracking Systems.
- AI Bullet Point Generation: Use AI tool to generate quantifiable bullet points from user inputs.
- AI Professional Summary: Automatically generate compelling summaries based on user's career stage.
- Keyword Suggestions: Suggest relevant keywords extracted from job descriptions.
- Multi-Format Export: Export resumes in PDF, DOCX, and TXT formats.
- Client-Side Data Handling: Data is stored only in the browser and permanently deleted on exit.

## Style Guidelines:

- Primary color: Deep Blue (#274472) to convey trust and professionalism.
- Secondary color: Air Force Blue (#5885AF) complements the primary color.
- Background color: Off White (#F8F8F8) provides a clean, readable backdrop.
- Light color: Light Cyan (#C3E0E5) gives a modern and softer touch.
- Text color: Charcoal Black (#1B1B1B) ensures readability and a professional feel.
- Accent color: Accent Orange (#E55934) to highlight key elements and calls to action.
- Headline font: 'Poppins' (sans-serif) for a contemporary, precise feel.
- Body font: 'PT Sans' (sans-serif) combining modern look and a little warmth or personality
- Clean and minimal layout with a strong emphasis on readability and clear visual hierarchy.
- Simple, professional icons to enhance usability without being distracting.
